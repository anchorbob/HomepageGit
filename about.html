  • What I did

    • So far, my achievements are on three parts. They are course and intern projects, being teaching assistant four times, and winning some awards both in academic and contests

      1. Projects

        Crio9229 & Crio9234 Implementation and Test
        During the summer intern in National Instrument, I was responsible for implementing the interface between compact reconfigurable I/O device(CRIO) 9229 & 9234 and the chassis FPGA.
      • Developed in a bottom-up fashion, from individual component to top level digital hardware design in VHDL
      • Tested and simulated in ModelSim
      • Developed plug-ins in Labview and tested the interface on real hardware

        Designed a synthesizable out-of-order processor in Verilog
        In course EECS470, I implemented an out-of-order arbitrary-way super-scalar processor with 4 other teammates using system Verilog with the following features:
      • P6 scheme
      • Works for 2-8 arbitrary way super-scalar by customizing parameters
      • Data forwarding from stores to loads(lsq) so as to issue load and store out of order
      • Non blocking cache
      • 2-bit local history predictor with BTB

        Smart car with an adaptive cruise control system based on visual and haptic feedback
        In course EECS461，I built an Adaptive Cruise Control system using the MPC5643L architecture which operated over CAN network with one other team mate. The cruise control system has the ability to navigate around curves obstacles, and other cars on the road
      • Created a Simulink module that models a simple vehicle with CAN bus and implemented an adaptive cruise control system
      • Implemented an automatic steering controller using a PID implementation to keep the vehicle at the center of the road

        Face detection based on Adaboost algorithm in Matlab
        In Fall 2014, I implemented a face detection program based on Adaboost algorithm in Matlab. It Trained a single Adaboost classfier with 50 weak-learners on provided data. It Achieved more than 99% accuracy of face detection for trained data and 95% for untrained data

        Fabrication and test of IC devices on an 8 inch silicon wafer
        In Fall 2014, I fabricated and tested poly-silicon gate, MOSFET, diode, capacitor, and etc. devices on an 8 inch wafer, during which, I gained valuable hand-on experience with RCA clean, photolithography, dry and wet etching, plasma ashing, thermal treatments, CVD and PVD manufacturing techniques in Lurie Nanofabrication Facility, which is one of the best clean room in US.

        J-EYE: Head Mounted Display Device
        Under the guide of Dr. Wenjie Wan, we developed a novel head mounted display device named J-EYE.

        Human Interface Device Design with PIC32 Board
        In course Ve373, I designed a human interface device with PIC32 board using UART communication protocol. Using it, users can modify the text displayed on LCD in real time.
